# Guide Architecture Hexagonale

Guide pratique pour appliquer l architecture hexagonale (Ports & Adapters) dans une application TypeScript/Next.js.

## Objectifs

- Decoupler la logique metier des details techniques (DB, API, framework).
- Expliquer les roles des ports entrants/sortants et des adapters.
- Montrer une implementation concrete avec use case testable.
- Fournir une demo interactive pour visualiser le swap d adapters.

## Structure des sections

1. Introduction
2. Principes fondamentaux
3. Schema Ports & Adapters
4. Flux d execution d un use case
5. Implementation Next.js
6. Migration progressive
7. Tests et maintenabilite
8. Anti-patterns
9. Demo interactive

## Prerequis

- TypeScript intermediaire
- Bases de Next.js App Router
- Notions de separation des responsabilites
