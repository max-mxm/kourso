# Guide Zod -- Validation TypeScript-first

Guide complet sur Zod, la librairie de validation de schemas TypeScript-first.

## Contexte

Ce guide est base sur l'experience d'utilisation de Zod dans le SaaS [Scanorr](https://scanorr.app), principalement pour la validation d'objets API et de formulaires. Il couvre les fondamentaux jusqu'aux patterns avances de production.

- TypeScript (notions de base)
- React / Next.js (pour les sections integration)
- Notions de validation de donnees
